# abanda-online-GEOG678-spring2026
# GEOG 678 – Lab 6 (Adan Banda)

## Overview
This lab focuses on using Leaflet with GeoJSON data to create interactive web maps. It includes working with different geometry types, styling features, and implementing interactivity such as hover effects and click events.



## Maps Included

### 1. Tutorial 1 – GeoJSON
- Demonstrates how to load and display GeoJSON data
- Includes point, line, and polygon features
- Point feature contains an interactive popup



### 2. Tutorial 2 – Choropleth Map
- Demonstrates thematic mapping using GeoJSON data
- Features are styled based on attribute values
- Includes hover highlighting and click-to-zoom functionality
- Displays dynamic information using an info control



### 3. Custom TAMU Map
- Displays Texas A&M University building data from an external GeoJSON file
- Buildings are styled using TAMU-inspired colors
- Interactive features include:
  - Hover highlighting
  - Dynamic info panel showing building name and abbreviation
  - Click-to-zoom functionality
